JavaScript FAQ

var is function-scoped and can be redeclared. let is block-scoped and cannot be redeclared in the same scope. const is also block-scoped but cannot be reassigned.

map() returns a new array with transformed elements. forEach() iterates over elements but does not return a new array. filter() returns a new array with elements that pass a condition.

Arrow functions do not have their own this context; they inherit this from the surrounding scope. They also have a more concise syntax.

A Promise represents a value that may be available now, later, or never. It has three states: pending, resolved, or rejected, and supports chaining with .then() and error handling with .catch().

A closure is a function that retains access to its outer scope even after the outer function has executed. This allows variables to persist in memory and enables data encapsulation.
